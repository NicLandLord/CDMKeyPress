# CDM KeyPress

CDM KeyPress adds responsive visual key press feedback to supported cooldown icons on World of Warcraft Retail.

Built to stay lightweight and practical, it helps make ability presses easier to read at a glance without changing the core layout of your UI.

## Features

- Visual press feedback on supported cooldown icons
- Bundled `LibCustomGlow-1.0` backend for stable in-game glow rendering
- Multiple glow styles: `Button`, `Pixel`, `Autocast`, and `Proc`
- Minimap button with quick access menu
- Manual scan trigger from the menu or minimap button
- Profile-aware settings
- Lightweight Retail-focused runtime

## Current Support

- BetterCooldownManager
- Blizzard default UI

## Commands

- `/cdmkp menu`
- `/cdmkp scan`
- `/cdmkp status`
