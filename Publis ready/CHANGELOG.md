# Changelog

## 1.0

- First public beta release of CDM KeyPress.
- Added visual key press feedback for supported cooldown icons.
- Added a minimap button with left-click menu access and right-click scan.
- Added profile-aware runtime configuration.
- Added support for BetterCooldownManager.
- Added support for Blizzard default UI.
- Bundled LibCustomGlow-1.0 for a reliable glow backend.
- Added configurable glow types: Button, Pixel, Autocast, and Proc.
- Added localization support for English, French, Russian, Italian, and Spanish (esES).
- Added addon icon metadata for WoW UI integration.
- Refactored the runtime into modular core files for easier maintenance.
